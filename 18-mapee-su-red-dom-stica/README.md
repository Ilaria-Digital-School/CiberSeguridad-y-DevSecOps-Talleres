# Mapee su red doméstica

**Recordatorio teórico** Antes de comenzar este taller, repasemos los conceptos fundamentales que necesitará aplicar. Una red es un conjunto de dispositivos (hosts) interconectados mediante enlaces físicos o inalámbricos, capaces de intercambiar datos en forma de paquetes. Un host es cualquier dispositivo que envía o recibe tráfico: su ordenador, su smartphone, su impresora o su router. El enlace es el canal por el que viajan los datos (cable Ethernet, señal Wi-Fi, fibra óptica). El paquete es la unidad básica de datos transmitida en la red; contiene información de origen, destino y la carga útil. Un error frecuente en principiantes es confundir el router con el módem: el módem conecta su hogar al proveedor de internet (ISP), mientras que el router distribuye esa conexión entre los dispositivos internos y asigna direcciones IP locales (del tipo 192.168.x.x). Otro error habitual es ignorar que existen dispositivos invisibles a simple vista en la red, como televisores inteligentes, bombillas conectadas o consolas de videojuegos. Mini-ejemplo mental: imagine su red doméstica como un edificio de oficinas. El router es la recepción central, cada dispositivo es una oficina con su propio número (dirección IP), y los paquetes son sobres que circulan por los pasillos (enlaces). Conocer el plano del edificio es el primer paso para identificar puertas mal cerradas, es decir, vulnerabilidades.
